# Omar Emad Portfolio

Building beautiful & performant web experiences.

## Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **UI Components:** [Aceternity UI](https://ui.aceternity.com/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **3D:** [Spline](https://spline.design/)

## Features

- **Responsive Design:** Optimized for all devices.
- **Interactive UI:** Smooth animations and transitions.
- **Contact Form:** Integrated with Nodemailer for direct communication.
- **Modern Effects:** Custom cursor, spotlight effects, and parallax scrolling.

## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/omaremad212/Omaremad.git
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Set up environment variables:**
   Create a `.env.local` file with the following:
   ```env
   EMAIL_USER=your-email@gmail.com
   EMAIL_PASS=your-app-password
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```
